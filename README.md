## These scripts were used to analyze data and make figures for the paper "IFN-γ- GranzymeB + Natural killer cells are induced by IV BCG vaccination and associated with protection against tuberculosis in rhesus macaques": https://www.sciencedirect.com/science/article/pii/S1933021925001163
### Figure 1B follows analysis described by Nowicka et al 2017 (DOI: 10.12688/f1000research.11622.3) and requires fcs files which can be fount through fairdomhub: https://fairdomhub.org/studies/1338, it also requires metadata and panel description files to be used for differential analysis, these can be found under the text folder.
### All other figures included in this repository have a coresponding r script and text file
